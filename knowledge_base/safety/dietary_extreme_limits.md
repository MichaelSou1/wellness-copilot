# 饮食极端限制红线

## 热量底线（非医疗监督下）
- 成年女性：每日热量摄入不得低于 **1200 kcal**
- 成年男性：每日热量摄入不得低于 **1500 kcal**
- 任何低于此阈值的方案必须在草稿中明确"需医生 / 注册营养师监督"。

## 蛋白质上限提醒
- 健康成人长期超过 **2.2 g/kg 体重** 的蛋白质摄入建议提醒肝肾负担与水分平衡。
- 已知肝肾基础疾病的用户不得给出超出常规的高蛋白方案。

## 处方与诊断边界
- 任何健康类回答不得给出：
  - 具体药物名称与剂量
  - 疾病诊断结论
  - "服用 XX 即可改善 YY" 这类治疗承诺
- 涉及补剂时只能描述常见类别，并强调"请咨询医生 / 药师"。

## 慢病人群运动安全
- 冠心病、糖尿病、未控制高血压用户被建议高强度间歇训练（HIIT）、Sprint 等高心率运动 → 必须先要求获得心内科 / 内分泌科许可。
- 运动方案应优先低中强度有氧 + 抗阻，循序渐进。
