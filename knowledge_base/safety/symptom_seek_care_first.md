# 症状描述必须先导向就医的清单

以下症状一旦在用户描述中出现，回答必须**首句直接建议就医**，再考虑提供任何辅助信息：

## 心血管类
- 胸痛、胸闷、左肩或下颌放射痛 → 急诊评估
- 静息心率持续偏高（>100）或异常跳动（漏拍、心悸） → 心内科评估
- 运动后短时间内血压异常波动 → 心内科

## 神经类
- 头晕、眩晕、晕厥或近似晕厥 → 急诊或神经内科
- 肢体麻木、刺痛、单侧无力 → 神经内科

## 骨关节类
- 持续疼痛超过 72 小时不缓解 → 骨科 / 运动医学
- 明显肿胀且不消退 → 骨科
- 关节弹响伴疼痛或卡顿 → 运动医学

## 呼吸类
- 静息呼吸困难、喘鸣 → 呼吸科或急诊
- 运动诱发的反复哮喘样发作 → 呼吸科

## 慢性 / 全身
- 无原因体重快速下降（一月 >5%） → 内科
- 持续发热超过 3 天 → 内科
- 长期失眠（>2 周）且影响日间功能 → 精神科 / 睡眠门诊

## 写作结构（强制）
1. **首句**：明确建议就医（"请尽快就医 / 安排门诊"）
2. **次句**：说明就医方向（科室）
3. **辅助**：可以补充缓解策略，但不能替代就医建议
4. **禁止**：不得以"可能原因 + 进一步问诊"的方式替代就医
